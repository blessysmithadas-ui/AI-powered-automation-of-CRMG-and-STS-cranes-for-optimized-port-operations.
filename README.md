Advanced AI-Integrated Automation of CRMG and STS Cranes for Optimized Port Operations 

Overview:
This project presents an AI-enabled automation framework for Ship-to-Shore (STS) and Cantilever Rail-Mounted Gantry (CRMG) cranes used in container terminal operations. Inspired by modern port automation requirements, the system integrates Computer Vision, Artificial Intelligence, IoT, and embedded control systems to improve container handling efficiency, operational safety, and cargo throughput.

The solution was developed as a B.Tech Capstone Project and implemented as a working prototype demonstrating autonomous crane operation using real-time visual feedback and intelligent control systems.

Objectives:
─ Automate STS and CRMG crane operations
─ Improve precision in container handling
─ Reduce human intervention
─ Enhance operational safety
─ Enable intelligent container tracking
─ Minimize operational delays
─ Improve terminal productivity

Key Features:
─ AI-assisted container recognition
─ Real-time visual monitoring
─ Automated crane movement control
─ Autonomous pick-and-place operation
─ Embedded control architecture
─ Data logging and monitoring
─ Prototype implementation of intelligent crane automation
