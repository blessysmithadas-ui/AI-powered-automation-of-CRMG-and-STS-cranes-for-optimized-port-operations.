Project Overview: 
This project focuses on automating Ship-to-Shore (STS) and Cantilever Rail-Mounted Gantry (CRMG) cranes using Artificial Intelligence, Computer Vision, and IoT technologies. The system is designed to improve container handling efficiency, reduce human intervention, enhance operational safety, and optimize cargo movement within container terminals.

Objectives: 
Automate STS and CRMG crane operations
Improve precision in container handling
Enable AI-based container identification
Reduce operational delays
Improve port throughput and safety

Key Features: 
AI-assisted container recognition
Automated pick-and-place operations
Real-time visual tracking
Intelligent decision support
Data logging and monitoring
Prototype implementation of autonomous crane operation

Development Phases: 
Phase I: Manual prototype development
Phase II: Semi-automation using AI models, including YOLO and CNN
Phase III: Fully automated prototype implementation using Vortex 2.O
